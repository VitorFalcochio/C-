# 🚀 Meus Primeiros Passos em C#

Bem-vindo ao meu repositório de aprendizado! Comecei a estudar C# e .NET recentemente e aqui estou registrando a minha evolução na programação.

## 💻 Projetos neste Repositório

Atualmente, este repositório guarda os meus primeiros testes práticos com a linguagem:

### 1. Sistema de Controle de Despesas (Em desenvolvimento)
Um gerenciador financeiro de linha de comando (Console). 
- **Conceitos aplicados:** Orientação a Objetos (criação da classe `Despesa`), manipulação de strings e persistência de dados.
- **Status:** O programa já recebe os inputs do usuário e salva os gastos em um arquivo `.txt` (formato CSV) para não perder os dados ao fechar.

### 2. Jogo de Adivinhação
Um clássico jogo de lógica onde o usuário precisa adivinhar um número secreto gerado pelo computador (de 1 a 100).
- **Conceitos aplicados:** Laços de repetição (`while`), estruturas condicionais (`if/else`), geração de números aleatórios (`Random`) e conversão segura de dados (`TryParse`).

## 🏃‍♂️ Como rodar este projeto

Caso queira testar na sua máquina, você precisará ter o [.NET SDK](https://dotnet.microsoft.com/download) instalado.

1. Clone este repositório:
   ```bash
   git clone [https://github.com/VitorFalcochio/C-.git](https://github.com/VitorFalcochio/C-.git)
   

2.  Entre na pasta do projeto.
3.  Execute o comando no terminal:
    ```bash
    dotnet run
    ```
